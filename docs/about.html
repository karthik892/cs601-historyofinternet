About Page

Our assignment was to create a Website for Internet History. We had to summarize the videos given by the Professor William, the summary of each video should be about 2 pages double-spaced.

We have added images to better the user experience.

Each page is linked to their respective file, for example, About tab is linked to "about.html", we have created the following webpages for this project index.html, about.html, tsi-browser-wars.html, tsi-dot-com-bubble.html, history-of-internet.html, tsi-people.html, tsi-internet-search.html.

The goal of this project is to learn editing codes and how to make website with pycharm/Git.
